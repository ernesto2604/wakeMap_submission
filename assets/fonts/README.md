# SF Pro Font Assets

Place your licensed SF Pro font files in this folder.

This project is prepared to use two families:
- SF Pro Text
- SF Pro Display

Important:
- Do not rename files arbitrarily once declared in pubspec.yaml.
- Declare only files that actually exist.
- Map each declared file to the correct weight/style.

After adding files, update pubspec.yaml `flutter/fonts` with exact file names.
Then set `AppTypography.useBundledSfPro = true` in `lib/app/app_typography.dart`.
